# Ciszu Network

Página corporativa de **Ciszu Network** — innovación digital y desarrollo de software de alto rendimiento.

Liderada por **Ciszuko Antony**, CEO y fundador.

## Tecnologías

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Zustand

## Desarrollo

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Desplegado en Vercel.
